0、接收用户问题并对其优化改写

1、从数据库中获取可用的表
2、确定哪些表与问题相关
3、获取相关表的结构信息
4、根据问题及表结构信息生成查询语句
5、使用大型语言模型检查查询中常见错误
6、执行查询并返回结果
7、根据数据库引擎反馈修正错误直至查询成功
9、根据结果生成回答
10、输出图形化报表
